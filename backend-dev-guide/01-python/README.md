# 01 - Python 编程基础

AI 方向后端开发的首选语言。本章从零开始，目标是让你**能看懂 90% 以上的常见 Python 项目代码**。

## 目标读者

完全零基础的后端开发人员。不需要任何编程经验，但需要有基本的电脑操作能力和命令行使用意愿。

## 不在本章讨论范围

以下内容对入门阶段来说过于深入，明确排除：

- Metaclasses、descriptor protocol
- C extensions、ctypes
- GIL 内部原理（只需知道"它存在"）
- 高级 asyncio 模式（自定义 event loop、协程链等）
- `abc` 模块深入用法
- 单元测试（属于后续 [05 - Git 与协作](../05-git/) 或独立章节的内容）
- Windows 环境（本章仅覆盖 macOS 和 Linux）
