# AI 产品经理技术知识指南

## 这份指南是写给谁的？

如果你是一名产品经理，正在（或即将）负责**大语言模型（LLM）或图像生成模型的部署与服务化**方向的产品，这份指南就是为你准备的。

你不需要能写 CUDA kernel，也不需要推导 attention 的数学公式。但你需要建立足够的技术直觉，能够：

- 与工程团队高效沟通，不被术语吓住
- 理解技术选型背后的 trade-off，做出合理的产品判断
- 知道哪些参数该暴露给用户、哪些该藏起来

## 与 vllm-help 的关系

本目录（`basic-concepts/`）是**概念入门**，帮助你建立对 AI 基础设施的整体认知。

如果你需要了解 vLLM 推理框架的具体配置参数，请参考 [`vllm-help/`](../vllm-help/README.md)——那是一份面向实操的参数参考手册，按 12 个功能模块组织，每个参数都标注了暴露建议（普通用户 / 高级用户 / 平台管理）。

**建议的阅读顺序**：先通读本目录建立概念框架，再按需查阅 `vllm-help/` 中的具体参数。
